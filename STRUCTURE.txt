KEY FEATURES:
✅ Multiplayer (up to 6 players)
✅ Real-time combat (2-second ticks)
✅ Command queuing (WASD movement)
✅ Economy system (collect coins, buy stuff)
✅ 3 Abilities (explosion, shoot, shockwave)
✅ Up to 5 bots per player
✅ Collision damage system
✅ No friendly fire
✅ Beautiful ANSI terminal graphics
✅ Pure Node.js (no npm dependencies!)

TECH STACK:
- Node.js built-in modules only:
  • net (TCP sockets)
  • readline (keyboard input)
  • All game logic custom-built
